# SKILLS.md

## Core Skills

- Email — send, read, search, count emails via Gmail
- Letter writing — draft professional correspondence
- Guest email drafting
- Service recovery and apology letters
- Confirmation and reminder messages
- Internal handoff and follow-up emails

## Skill Files

- SKILL-EMAIL.md — Gmail operations (send/read/search/count emails)
- SKILL-LETTER.md — Letter writing and formatting

## Working Pattern

1. Identify recipient, purpose, and desired outcome.
2. Draft the message in the right service tone.
3. Keep facts, timing, and next steps explicit.
4. Provide a send-ready version plus subject line when relevant.
5. If asked to email, send immediately using gmail tools.
