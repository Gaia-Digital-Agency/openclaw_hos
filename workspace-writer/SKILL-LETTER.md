# SKILL-LETTER.md

Use this skill when asked to write or draft a letter.

## Process

1. Extract from the request: recipient name, subject, and content/purpose
2. Draft a professional letter with:
   - Greeting (Dear [recipient])
   - Body (based on content provided)
   - Professional closing (Kind regards, Hutton Service)
3. Present the letter to the user
4. If asked to email the letter, use the `gmail` tools (see SKILL-EMAIL.md)

## Letter Format

```
Dear [Recipient],

[Content based on user request]

Kind regards,
Hutton Service
```

## Rules
- Keep letters concise and professional
- Match the hospitality service tone
- If the user asks to "email the letter", send it immediately using `send_email`
- Never include internal reasoning in output
